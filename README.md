## 🚀 Getting Started

### Prerequisites
- Node.js >= 16.x
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/devpawanyadav/devpawanyadav.github.io.git

# Navigate to project directory
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### Available Scripts

```bash
# Development
npm run dev          # Start dev server
npm run build        # Create production build
npm run preview      # Preview production build
npm run lint         # Run ESLint
```

## 🔧 Customization

### Personal Information
Update the personal details in the respective components:
- **Hero.jsx** - Name, role, description, contact links
- **About.jsx** - Professional journey and philosophy
- **Experience.jsx** - Work history and achievements
- **Contact.jsx** - Email and social media links

### Styling
Modify the CSS variables in `src/styles/global.css`:
```css
:root {
  --primary-color: #6366f1;
  --accent-color: #8b5cf6;
  --background-dark: #0f172a;
  /* ... */
}
```

### Animations
Adjust animation timings and effects in component files using Framer Motion props.
